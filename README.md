Deerwallet
=====

Deerwallet is an application Tetra based to create your own blockchain MultiToken.

Install
-------

```
git clone https://github.com/Cryptodeer/deerwallet
cd deerwallet
sh install
```


Usage
-----

```
# start stop
dnet < start | stop >

# help
dnet help

# restart openresty
service dnet restart
```
